蓝奏云提示开通会员才能下载？
【说明】
蓝奏云禁止了非会员分享APK文件，下载时会提示 '非会员不在支持分享 apk 文件，请先开通会员' (此处是指分享文件的上传者去单独开通蓝奏云会员)

【解决方法】
在浏览器中将 UA (User Agent) 切换为 PC/桌面端/电脑模式 等即可正常下载
为什么在暮光内下载的文件无法找到？
请检查您是否给予了暮光读写本地文件的权限，一般下载文件默认存储于 'Download/暮光' 文件夹内，如果在文件管理中没有找到，可以通过授予暮光通知权限，并确保默认内置下载器为首选，下载进度会于状态栏中显示通知，下载进度可通过状态栏进行查看
为什么安装暮光的时候会报毒/风险/无法安装？
由于暮光内的功能大多数来源于网络收集或者用户投稿分享等，这些功能可能会存在一些问题。而且暮光本体是由 Kotlin Java luaJava 进行混编，无法正常上架各大手机厂商的应用市场，属于第三方应用，所以会在安装时会提示风险。如果无法安装，请暂时关闭系统中的纯净模式/安全守护等，或关闭网络后临时授权安装
暮光内功能侵权或违规怎么处理？
由于暮光中的大多数功能来源于网络收集或由用户投稿分享等，因此不可否认，这可能会无意间侵犯您或您公司合法的权利和利益。为此，我们表示诚挚的歉意。如果您对任何受影响的内容拥有权利，请通过以下方式正式提出申诉：发送电子邮件至 QiSHuHe@88.com，或填写侵权投诉问卷表单进行反馈。在收到您的通知后，开发者将在一个工作周内对相关功能进行下架处理。对此造成的任何不便，我们深感抱歉
